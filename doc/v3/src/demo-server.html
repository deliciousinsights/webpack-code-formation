La partie Webpack Dev / HMR

const webpack = require('webpack') const devMiddleware = require('webpack-dev-middleware') const hotMiddleware = require('webpack-hot-middleware') const config = require('../webpack.config')()

Vu qu’on est mono-main, on enrobe dans un tableau et on ajoute le client HMR (pas besoin quand on sert avec webpack-dev-server, mais là si, du coup).

config.entry.main = [...config.entry.main, 'webpack-hot-middleware/client'] const compiler = webpack(config) app.use(devMiddleware(compiler, { logLevel: 'warn' })) app.use(hotMiddleware(compiler))
